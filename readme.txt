This plugin brings an accordion faq block to the new WordPress Gutenberg editor.

== Description ==

Easily add a Accordion FAQ Block to your WordPress post or page via the new Gutenberg Editor.

- Add Title
- Add Text


== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Accordion FAQ"
3. **Install** the "Accordion FAQ  Block" plugin
4. **Activate** "Accordion FAQ" from your Plugins page
5. **Insert Block** on your Gutenberg Editor and select "Accordion FAQ"
